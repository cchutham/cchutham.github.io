WELCOME TO HELP CENTER!

Numbers and operator signs shown in calculator can be used in any combination. For example, if you have button 2 and 5, you can create numbers such as 255, 522, 525 etc.

Buttons Explanation
"=" : Equal sign. This button will show the total result after number operation(s).
"AC": To clear the screen, but does not erase numbers in memory.
"M+": To add the value to memory.
"M-": To subtract the valur from the memory.
"MRC": To recall and clear the memory.
